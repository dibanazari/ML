# CO2 Emissions Prediction using Multiple Linear Regression

## Overview

This project applies **Multiple Linear Regression** to predict vehicle **CO2 emissions** using vehicle characteristics and fuel consumption data. The model was developed in Python using Scikit-learn and evaluated with common regression performance metrics.

This project was completed as part of my Machine Learning learning journey.

---

## Dataset

The dataset contains information about vehicles, including:

- Engine Size
- Number of Cylinders
- Combined Fuel Consumption
- CO2 Emissions (Target Variable)

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## Project Workflow

1. Load and explore the dataset
2. Visualize relationships between features and CO2 emissions
3. Split the dataset into training and testing sets
4. Train a Multiple Linear Regression model
5. Predict CO2 emissions
6. Evaluate the model using:
   - Mean Squared Error (MSE)
   - R² Score

---

## Features Used

- Engine Size
- Cylinders
- Combined Fuel Consumption

Target:

- CO2 Emissions

---

## Results

The model demonstrates how multiple vehicle characteristics influence CO2 emissions and provides a practical example of applying supervised machine learning for regression problems.
